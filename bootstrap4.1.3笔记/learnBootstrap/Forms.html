Vertical (basic) form

Form controls automatically receive some global styling with Bootstrap: All textual <\input>, <\textarea>, and <\select> elements with class .form-control have a width of 100%.

Standard rules for all three form layouts: Always use <\form role="form"> (helps improve accessibility for people using screen readers) Wrap labels and form controls in <\div class="form-group"> (needed for optimum spacing 需要最佳间距) Add class .form-control to all textual <\input>, <\textarea>, and <\select> elements

Inline form

Note: This only applies to forms within viewports that are at least 768px wide! 这仅适用于至少76px宽的视口内的表单！ Additional rule for an inline form: Add class .form-inline to the <\form> element

Make the viewport larger than 768px wide to see that all of the form elements are inline, left aligned, and the labels are alongside.

Horizontal form!!!

A horizontal form stands apart from the other forms both in the amount of markup, and in the presentation of the form. 水平形式与其他形式在标记的数量和表单的呈现上都是分开的。 Additional rules for a horizontal form: Add class .form-horizontal to the <\form> element Add class .control-label to all <\label> elements Tip: Use Bootstrap's predefined grid classes to align labels and groups of form controls in a horizontal layout.
